Glosario de IA

Aprende los conceptos fundamentales de Inteligencia Artificial, Machine Learning y desarrollo moderno de forma clara y visual.

🤖 Inteligencia Artificial y Machine Learning

Inteligencia Artificial

Sistemas capaces de realizar tareas que normalmente requieren inteligencia humana como razonar o aprender.

Machine Learning

Rama de la IA que permite a las máquinas aprender patrones a partir de datos.

LLM

Modelos de lenguaje entrenados con grandes cantidades de texto para comprender y generar lenguaje humano.

Prompt

Instrucción o pregunta que un usuario entrega a un modelo de IA.

Algoritmo

Conjunto de pasos o reglas que una computadora sigue para resolver un problema.

Deep Learning

Machine Learning basado en redes neuronales profundas capaces de analizar datos complejos.

RAG

Técnica donde un modelo consulta documentos externos antes de generar una respuesta.

Fine‑tuning

Entrenamiento adicional de un modelo para especializarlo en un tema específico.

🌐 Desarrollo Web

HTML

Lenguaje que define la estructura de una página web.

CSS

Lenguaje encargado del diseño visual y estilos.

JavaScript

Lenguaje que permite interacción y lógica en aplicaciones web.

Frontend

Parte visible con la que interactúa el usuario.

Backend

Parte lógica del servidor que procesa datos y APIs.

Framework

Conjunto de herramientas que facilitan el desarrollo de software.

🚀 Infraestructura y Herramientas

Plataformas de IA

Ecosistemas que ofrecen modelos y herramientas para crear aplicaciones de inteligencia artificial.

API

Interfaz que permite que diferentes aplicaciones se comuniquen entre sí.

Token

Unidad mínima de texto que procesan los modelos de lenguaje.

Alucinación

Cuando una IA genera información incorrecta que parece real.

Base de Datos Vectorial

Base optimizada para buscar información por significado y similitud.

💡 Conceptos Emergentes y Ética

IA Generativa

Tecnología capaz de crear contenido nuevo como texto, imágenes o código.

AGI

Hipotética inteligencia artificial con capacidades cognitivas equivalentes a las humanas.

Agente de IA

Sistema capaz de ejecutar acciones de forma autónoma para cumplir objetivos.

Sesgo Algorítmico

Prejuicios que un modelo puede heredar de los datos de entrenamiento.

Parámetros

Variables internas que el modelo ajusta durante su entrenamiento.
