Inteligencia Artificial y Machine Learning

Inteligencia Artificial (IA)

Campo de la informática dedicado a crear sistemas capaces de realizar tareas que normalmente requieren inteligencia humana, como el razonamiento, el aprendizaje y la resolución de problemas.

Machine Learning

Una rama de la IA que se centra en el uso de datos y algoritmos para imitar la forma en que los humanos aprenden, mejorando gradualmente su precisión sin ser programados explícitamente.

LLM

Large Language Model. IA entrenada con cantidades masivas de texto para comprender, generar y manipular lenguaje humano de forma coherente.

Prompt

La instrucción, pregunta o texto que un usuario proporciona a un modelo de IA para obtener una respuesta específica. Determina la utilidad del resultado.

Algoritmo

Un conjunto de reglas o pasos ordenados que una computadora sigue para realizar una tarea. En IA, procesan datos para encontrar patrones.

Deep Learning

Evolución del Machine Learning que utiliza redes neuronales artificiales con muchas capas para procesar datos complejos como imágenes o voz.

RAG

Retrieval-Augmented Generation. Técnica que permite a una IA consultar fuentes externas de datos en tiempo real antes de generar una respuesta, reduciendo alucinaciones.

Fine-tuning

Proceso de tomar un modelo de IA ya pre-entrenado y entrenarlo un poco más con un conjunto de datos específico para que se vuelva experto en un tema.

Desarrollo y Tecnologías Web

HTML

El lenguaje de marcado estándar para crear la estructura de las páginas web. Es el "esqueleto" que define dónde van los títulos, párrafos e imágenes.

CSS

El lenguaje de diseño encargado de la apariencia visual de una web. Es la "piel" y la "vestimenta" del esqueleto HTML.

JavaScript (JS)

Lenguaje que permite hacer páginas web interactivas. Se encarga del comportamiento, animaciones y la comunicación con servicios de IA.

Frontend y Backend

El Frontend es la parte visual (HTML/CSS/JS). El Backend es la parte lógica en el servidor, donde suelen procesarse las llamadas pesadas a la IA.

Framework

Un conjunto de herramientas y reglas predefinidas que ayudan a escribir código de forma rápida (ej: React para frontend, FastAPI para backend).

🚀

Infraestructura y Herramientas

Plataformas de IA

Ecosistemas que ofrecen herramientas, modelos y servicios para implementar soluciones de IA rápidamente (ej: OpenAI, Hugging Face).

API

Un puente que permite que dos aplicaciones se comuniquen entre sí, como una web consultando a un modelo de IA.

Token

La unidad básica que procesan los LLM. Un token puede ser una palabra o fragmento. Determina cuánto texto puede leer/escribir el modelo.

Alucinación

Término usado cuando una IA genera información que parece correcta pero es factualmente falsa o inventada.

Base de Datos Vectorial

Almacenamiento especializado que guarda información como números (vectores), permitiendo a la IA buscar por "significado" y no solo palabras clave.

Conceptos Emergentes y Ética

IA Generativa

Tecnología capaz de crear contenido original (texto, imágenes, audio, código) a partir de patrones aprendidos de datos existentes.

AGI

Artificial General Intelligence. Nivel teórico donde una máquina puede entender, aprender y realizar cualquier tarea intelectual humana.

Agente de IA

Sistema que no solo responde, sino que ejecuta acciones autónomas (buscar, usar herramientas) para cumplir un objetivo.

Sesgo Algorítmico

Prejuicios que una IA hereda de los datos de entrenamiento, pudiendo generar resultados injustos o discriminatorios.

Parámetros

Variables internas que el modelo "aprende". A mayor cantidad, mayor es la capacidad general del modelo para entender matices complejos.
