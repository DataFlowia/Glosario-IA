IA y Machine Learning

Inteligencia Artificial (IA)

Campo de la informática dedicado a crear sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.

Machine Learning

Uso de datos y algoritmos para imitar la forma en que los humanos aprenden, mejorando gradualmente su precisión.

LLM

Large Language Model. IA entrenada con cantidades masivas de texto para comprender y generar lenguaje.

Prompt

Instrucción o texto que un usuario proporciona a un modelo de IA para obtener una respuesta específica.

Algoritmo

Conjunto de reglas que una computadora sigue. En IA, procesan datos para encontrar patrones.

Deep Learning

Evolución del ML que utiliza redes neuronales artificiales profundas para procesar datos complejos.

RAG

Técnica que permite a una IA consultar fuentes externas de datos en tiempo real antes de responder.

Fine-tuning

Entrenamiento adicional de un modelo pre-entrenado con datos específicos para especializarlo.

Desarrollo y Tecnologías Web

HTML

Lenguaje de marcado estándar para crear la estructura o "esqueleto" de las páginas web.

CSS

Lenguaje de diseño encargado de la apariencia visual, colores y disposición de una web.

JavaScript (JS)

Lenguaje que permite interactividad y lógica dinámica en el navegador del usuario.

Frontend y Backend

El Frontend es lo visual; el Backend es la lógica del servidor y gestión de datos.

Framework

Herramientas y reglas predefinidas que aceleran el desarrollo de software.

🚀

Infraestructura y Herramientas

Plataformas de IA

Servicios que ofrecen modelos y herramientas listas para usar (ej: OpenAI).

API

Puente que permite que dos aplicaciones se comuniquen entre sí automáticamente.

Token

Unidad básica que procesan los LLM. Equivale aproximadamente a una palabra.

Alucinación

Cuando una IA genera información falsa pero gramaticalmente convincente.

Base de Datos Vectorial

Almacén especializado para que la IA busque información por "sentido" o concepto.

💡

Conceptos Emergentes y Ética

IA Generativa

Tecnología capaz de crear contenido original a partir de patrones aprendidos.

AGI

Nivel teórico de IA con capacidad intelectual general similar a la humana.

Agente de IA

Sistema autónomo que ejecuta acciones para cumplir un objetivo específico.

Sesgo Algorítmico

Prejuicios heredados de los datos que pueden causar resultados injustos.

Parámetros

Variables internas que definen el conocimiento y capacidad del modelo.
